# CIAOD  5 семестр

Репозиторий содержит лабораторные работы 4, 5, 6.

## Состав
- lab4.ipynb — интерполяция рациональными функциями беррута
- lab5.ipynb — аппроксимация и линеаризация функций
- lab6.ipynb — компьютерная геометрия

## Требования
- python 3.10+
- vscode с расширением jupyter
- пакеты: numpy, matplotlib

## Запуск
- откройте нужный файл .ipynb в vscode
- выберите python-ядро
- запустите ячейки
